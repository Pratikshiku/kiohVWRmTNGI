### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、指导教师、评阅教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 课题信息
6. 选题申请
7. 课题任务
8. 毕业论文
9. 公告信息
10. 评阅教师信息
11. 用户信息
12. 指导教师信息



**指导教师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 课题信息管理
6. 选题申请管理
7. 课题任务管理
8. 毕业论文管理
9. 公告信息管理
10. 用户管理



**评阅教师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 毕业论文管理
6. 公告信息管理
7. 用户管理




**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 管理员管理
4. 毕业论文管理
5. 课题信息管理
6. 选题申请管理
7. 课题任务管理
8. 基础数据管理（班级管理、课题类型管理、任务类型管理、公告类型管理、学院管理）
9. 公告信息管理
10. 评阅教师管理
11. 用户管理
12. 指导教师管理
